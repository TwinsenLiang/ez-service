# Changelog

所有重要的变更都会记录在此文件中。

版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)：MAJOR.MINOR.PATCH

## [1.0.0] - 2025-12-13

### 新增功能

#### 端口管理
- ✨ 启动前端口冲突检测
  - 支持 lsof/netstat/ss 多种检测方式
  - 显示占用进程详细信息
  - 清晰的错误提示和解决方案

- ✨ 停止时端口自动清理
  - 优雅停止后检查端口释放
  - 强制停止后清理残留进程
  - 确保端口完全释放

#### 进程清理
- ✨ 新增 `cleanup` 命令
  - 清理 PID 文件对应的进程
  - 清理端口占用的所有进程
  - 适用于僵尸进程和孤立端口

#### 服务管理
- ✅ 启动/停止/重启服务
- ✅ 服务状态查看（进程信息、CPU、内存、端口监听）
- ✅ 实时日志查看
- ✅ 测试程序运行

#### URL 显示
- 📡 启动时显示访问地址
  - 本地访问: http://localhost:端口
  - 局域网访问: http://IP:端口
- 📡 状态查询时显示访问地址
- 🔍 端口监听状态检测

#### 环境支持
- 🐍 Python 虚拟环境支持
- 📦 依赖自动检查和安装
- 🔧 多平台支持（macOS、Linux）

### 修复问题

- 🐛 修复 `restart` 命令执行 `stop` 后退出的问题（exit 改为 return）
- 🐛 修复端口配置说明不够清晰的问题

### 改进

- 📝 改进帮助信息，添加 cleanup 命令说明
- 📝 改进 SERVICE_PORT 配置注释
  - 添加 ⚠️ 提示 Web 服务必须设置
  - 添加常见端口示例
  - 说明非 Web 服务可保持为空

### 架构

- 🏗️ 确立"品牌标准+产品差异化"架构
  - 模板定义通用标准
  - 项目可独立差异化
  - 共性需求提升到模板

## 版本说明

### MAJOR 版本（破坏性变更）
- 不兼容的 API 变更
- 配置文件格式重大变更
- 命令参数重大调整

### MINOR 版本（新功能）
- 向后兼容的新功能
- 新增命令
- 功能增强

### PATCH 版本（修复）
- 向后兼容的问题修复
- 小的改进和优化
